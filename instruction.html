How to use the Bar Inventory Management System

Welcome to the Bar Inventory Management System. This guide will help you navigate and use the application effectively.

Main View

Upon starting the app, you will see two tabs: List View and Manage View.

List View

This is where you can manage and view your inventory levels. It is divided into three phases:

  • Start: Record initial inventory levels.
  • End: Record final inventory levels.
  • Difference: Display differences between start and end levels.

Buttons in List View:

  • Start: Switch to initial inventory.
  • End: Switch to final inventory.
  • Difference: Show differences between start and end levels.
  • Show Formula / Show Sum: Toggle between entering formulas and displaying sums.

Other functions in List View:

  • Save: Save your entries.
  • Export to CSV: Export data as a CSV file.
  • Export Inventory: Export data as a JSON file.
  • Import Inventory: Import data from a JSON file.

Manage View

Here you can manage categories, areas, and items.

Buttons and fields in Manage View:

  • Add New Category: Enter a name for a new category and click "Add Category" to add it.
  • Remove: Remove a category from the list.
  • Add New Area: Enter a name for a new area and click "Add Area" to add it.
  • Remove: Remove an area from the list.
  • Name: Enter the name of a new item.
  • Category: Select the category of the new item.
  • Add Item: Add the new item.

Summary of Buttons and Functions

  • Start: Record initial inventory levels.
  • End: Record final inventory levels.
  • Difference: Display differences between start and end levels.
  • Show Formula / Show Sum: Toggle between entering formulas and displaying sums.
  • Save: Save your entries.
  • Export to CSV: Export data as a CSV file.
  • Export Inventory: Export data as a JSON file.
  • Import Inventory: Import data from a JSON file.
  • Add New Category: Add a new category.
  • Remove: Remove a category or area.
  • Add New Area: Add a new area.
  • Add Item: Add a new item.
  • Edit: Edit an item.
  • Delete: Delete an item.

Anleitung zur Nutzung des Bar Inventory Management Systems

Willkommen beim Bar Inventory Management System. Diese Anleitung hilft Ihnen, die App effektiv zu nutzen.

Hauptansicht

Beim Starten der App sehen Sie zwei Registerkarten: List View und Manage View.

List View

Hier können Sie die Bestände Ihrer Artikel verwalten und einsehen. Diese Ansicht ist in drei Phasen unterteilt:

  • Start: Anfangsbestände erfassen.
  • End: Endbestände erfassen.
  • Difference: Differenzen zwischen Anfangs- und Endbeständen anzeigen.

Schaltflächen in der List View:

  • Start: Wechseln zu den Anfangsbeständen.
  • End: Wechseln zu den Endbeständen.
  • Difference: Anzeige der Differenzen zwischen Anfangs- und Endbeständen.
  • Show Formula / Show Sum: Wechseln zwischen der Eingabe von Formeln und der Anzeige von Summen.

Weitere Funktionen in der List View:

  • Save: Speichert Ihre Eingaben.
  • Export to CSV: Exportiert Ihre Daten als CSV-Datei.
  • Export Inventory: Exportiert Ihre Daten als JSON-Datei.
  • Import Inventory: Importieren Sie eine zuvor exportierte JSON-Datei, um Ihre Bestände wiederherzustellen.

Manage View

Hier können Sie Kategorien, Bereiche und Artikel verwalten.

Schaltflächen und Felder in der Manage View:

  • Add New Category: Geben Sie einen Namen für eine neue Kategorie ein und klicken Sie auf "Add Category", um sie hinzuzufügen.
  • Remove: Entfernt eine Kategorie aus der Liste.
  • Add New Area: Geben Sie einen Namen für einen neuen Bereich ein und klicken Sie auf "Add Area", um ihn hinzuzufügen.
  • Remove: Entfernt einen Bereich aus der Liste.
  • Name: Geben Sie den Namen eines neuen Artikels ein.
  • Category: Wählen Sie die Kategorie des neuen Artikels aus.
  • Add Item: Fügt den neuen Artikel hinzu.

Zusammenfassung der Schaltflächen und Funktionen

  • Start: Erfassen der Anfangsbestände.
  • End: Erfassen der Endbestände.
  • Difference: Anzeige der Differenzen zwischen Anfangs- und Endbeständen.
  • Show Formula / Show Sum: Wechsel zwischen Formel- und Summenansicht.
  • Save: Speichert Ihre Eingaben.
  • Export to CSV: Exportiert die Daten als CSV-Datei.
  • Export Inventory: Exportiert die Daten als JSON-Datei.
  • Import Inventory: Importiert Daten aus einer JSON-Datei.
  • Add New Category: Hinzufügen einer neuen Kategorie.
  • Remove: Entfernt eine Kategorie oder einen Bereich.
  • Add New Area: Hinzufügen eines neuen Bereichs.
  • Add Item: Hinzufügen eines neuen Artikels.
  • Edit: Bearbeiten eines Artikels.
  • Delete: Löschen eines Artikels.
